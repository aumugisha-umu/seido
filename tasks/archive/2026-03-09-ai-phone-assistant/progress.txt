## US-007: ElevenLabs webhook handler ✅

- **Learning**: ElevenLabs SDK v1.59 has NO webhooks.constructEvent() method — must implement manual HMAC-SHA256 verification. Format: t=timestamp,v0=hmac_hex, signed as SHA256(timestamp.body, secret). timingSafeEqual for comparison.

- **Decision**: Email recap reuses existing EmailNotificationService (eventType: 'created') — no need for separate email template (US-008 partially absorbed into US-007).
- **Decision**: Usage counter uses upsert pattern (check existing → update or insert) rather than ON CONFLICT because ai_phone_usage composite key (team_id, month) needs checking.
