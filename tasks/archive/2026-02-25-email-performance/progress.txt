## Key Learnings
- **Stale-while-revalidate at client level**: `useRef<Map>` with timestamp TTL is simpler than SWR/React Query for single-component caching. No library needed.
- **AbortController + cache = double protection**: Cache prevents re-fetch, abort prevents stale responses when cache misses.
- **head:true count queries vs bulk fetch**: PostgREST `head:true` counts rows server-side, transfers zero data. Always prefer over fetch + JS count.
- **SQL GROUP BY via RPC**: When you need aggregates, push to SQL. JS grouping of unbounded result sets is the #1 SSR perf trap.
