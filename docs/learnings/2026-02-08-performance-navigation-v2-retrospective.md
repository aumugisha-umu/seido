## Recommendations for Similar Future Work

1. **Before upgrading libraries**: Check the changelog for breaking changes
2. **After each file edit**: Run `npx tsc --noEmit` on that specific file
3. **For batch queries**: Always use Map for O(1) lookup, not filter/find
4. **For Suspense**: Create separate async component file for clean separation
5. **For virtual scroll**: Always add a threshold — don't virtualize small lists
